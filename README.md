# cpupower-gui-qt

A Qt-based fork of [cpupower-gui](https://github.com/vagnum08/cpupower-gui), the original GTK application for managing CPU frequency scaling.

<img width="984" height="1063" alt="screenshot" src="https://github.com/user-attachments/assets/742e15a6-7d7c-4fdf-ad35-56e6ac4abdf4" />

## About

This project is a complete rewrite of the user interface from GTK to Qt (PyQt6), while maintaining compatibility with the original cpupower-gui backend and configuration files.

### Modifications

- **UI Framework**: Rewritten from GTK/libadwaita to Qt (PyQt6)
- **Original Repository**: [vagnum08/cpupower-gui](https://github.com/vagnum08/cpupower-gui)
- **Fork Author**: Felipe Figueiredo ([@FelipeFMA](https://github.com/FelipeFMA)) — felipefmavelar@gmail.com
- **Fork Date**: December 2025
